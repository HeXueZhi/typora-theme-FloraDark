### 安装方法

1. 通过Typora的**设置->主题->打开主题文件夹**，来打开文件夹。
2. 将`flora文件夹`、`flora-hans.css`以及`flora-dark.css`复制到typora的主题文件夹下。
3. 重启Typora，就可以选择新的FloraDark主题了。
